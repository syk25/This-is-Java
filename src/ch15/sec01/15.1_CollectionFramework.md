# 15.1 컬렉션 프레임워크
자료구조를 바탕으로 객체들을 효율적으로 추가, 삭제, 검색할 수 있도록 관련 된 인터페이스와 클래스들을 총칭해서 Collection Framework 라고 한다.
해당 타입들은 java.util 패키지에 포함되어 있다.

## 인터페이스의 준위
- Collection(인터페이스)
  - List(인터페이스)
    - ArrayList
    - Vector
    - LinkedList
  - Set(인터페이스)
    - HashSet
    - TreeSet
- Map
  - HashMap
  - HashTable
  - TreeMap
  - Properties
